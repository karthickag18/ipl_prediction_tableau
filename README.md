# 🏏 IPL Analysis Dashboard | Tableau

## 📌 Project Overview

The **IPL Analysis Dashboard** is an interactive Tableau project that provides comprehensive insights into the Indian Premier League (IPL). It allows users to analyze team performances, player statistics, match outcomes, venue trends, and season-wise records through dynamic visualizations and filters.

This dashboard helps cricket enthusiasts, analysts, and sports professionals explore IPL data efficiently and make data-driven observations.

---

## 🚀 Features

* 📊 Season-wise IPL analysis
* 🏆 Match winner analysis
* 🥇 Orange Cap (Highest Run Scorer)
* 🟣 Purple Cap (Highest Wicket Taker)
* 🏏 Batting statistics dashboard
* 🎯 Bowling statistics dashboard
* 🌆 City-wise matches hosted
* 🏟️ Venue-wise match analysis
* 🪙 Toss decision vs match-winning percentage
* 💥 Tournament-wise total fours and sixes
* Interactive filters for season and teams
